Clique para copiar python -m http.server 8000

1. Abra o Prompt de Comando na pasta do arquivo
2. Execute o comando acima
3. Abra: http://localhost:8000

Não tem Python? Use PowerShell: npx serve .
Clique para copiar python3 -m http.server 8000

1. Abra o Terminal na pasta do arquivo
2. Execute o comando acima
3. Abra: http://localhost:8000

Clique para copiar (instala e serve) npx serve . -p 8000

Funciona se tiver Node.js instalado.
Depois abra: http://localhost:8000

Alternativa: Chrome Flag
Permite microfone em file:// sem servidor.

Cole na barra do Chrome e pressione Enter: chrome://flags/#unsafely-treat-insecure-origin-as-secure

Adicione file:// na lista e reinicie o Chrome.
